DNS GitHub Sync

Экспорт текущих domain-list групп в файл для GitHub и синхронизация роутера с raw-ссылкой. Формат текстовый, поэтому другой роутер может подтянуть тот же файл напрямую через GitHub.

GitHub источник

Поддерживаются raw-ссылки GitHub/Gist. Обычная ссылка вида github.com/user/repo/blob/main/file.txt будет преобразована в raw.

Отправить в GitHub

Кнопка ниже берёт актуальный снимок прямо с роутера и коммитит его в GitHub по raw URL выше. Нужен fine-grained token с правом Contents: Read and write для выбранного репозитория. Отправка работает для файла в репозитории GitHub; Gist можно использовать только как источник для чтения.

Текущий снимок роутера

Это содержимое можно положить в GitHub как обычный текстовый файл. В него входят название/описание группы, все include и текущий DNS-маршрут группы.

Группы в снимке

Короткая проверка того, что будет отправлено в GitHub: номера списков, описания, количество include и маршрут.
