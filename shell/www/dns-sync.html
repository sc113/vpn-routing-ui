DNS-группы

Один текстовый файл для переноса Keenetic domain-list групп между роутерами. Сначала получи актуальный снимок, затем вставь этот текст на другом роутере и примени его.

Готовим сохранение 0%

Текстовый код DNS

В тексте сохраняются названия групп, все include и назначенные DNS-маршруты. Применение полностью заменяет DNS-группы роутера содержимым поля.

Редактор DNS-групп

Клик по группе открывает её хосты. Хосты редактируются по одному в строке и сохраняются в тот же переносимый текст.
