DNS-группы

DNS-списки синхронизируются с GitHub по дате версии. Более новая сторона автоматически становится источником, а загруженный файл строит Keenetic domain-list группы без изменения ProxyN и DNS-маршрутов.

Автоматическая синхронизация

«Обновить» сравнивает версии до секунды. Новая версия роутера отправляется в GitHub, новая версия GitHub скачивается и применяется на роутере.

Версия GitHub Проверяем… Читаем дату последнего коммита файла
Источник sc113/vpn-rui-dns main · vpn-routing-ui-dns-groups.txt

Секрет хранится на роутере и никогда не возвращается в браузер.

Текстовый код DNS

В тексте сохраняются названия групп и все include. Формат v2 хранит названия обычным читаемым текстом. Применение добавляет или обновляет DNS-группы, но не меняет назначенные DNS-маршруты и ProxyN.

Редактор DNS-групп

Клик по группе открывает её хосты. Хосты редактируются по одному в строке и сохраняются в тот же переносимый текст.
