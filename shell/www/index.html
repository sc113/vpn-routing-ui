VPN Routing UI

Главная страница для управления VPN-маршрутами, движками и восстановлением роутерного слоя. Профили подключений, DNS-правила и состояние ProxyN собираются в одном интерфейсе.

Быстрый доступ

Только переходы на отдельные страницы конфигураций. Сервисные действия и обновления вынесены ниже.

Ядра и сервисы

У каждого движка свои кнопки управления сервисом и пакетом. Обновление через Entware идёт отдельно и только в безопасном режиме без prerelease.
